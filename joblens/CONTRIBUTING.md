# 贡献指南

感谢你对 JobLens 的关注！欢迎提交贡献。

## 如何贡献

### 报告Bug
1. 在 [Issues](https://github.com/ZBCs-StudioCr-CN/joblens/issues) 中提交
2. 标题格式：`[Bug] 简要描述问题`
3. 附上复现步骤和截图

### 提交功能建议
1. 在 [Issues](https://github.com/ZBCs-StudioCr-CN/joblens/issues) 中提交
2. 标题格式：`[Feature] 功能描述`
3. 说明使用场景和预期效果

### 提交代码
1. Fork 本仓库
2. 创建特性分支：`git checkout -b feature/your-feature`
3. 提交更改：`git commit -m "feat: 添加xxx功能"`
4. 推送分支：`git push origin feature/your-feature`
5. 提交 Pull Request

## 开发规范

- 纯前端项目，不引入构建工具
- JavaScript 使用 ES5+ 语法，兼容主流浏览器
- 中文注释
- 代码风格保持与现有代码一致
